In Anthropocene

What is In Anthropocene?

Anthropocene is a website that helps predict the severity of forest fires in California as you key in various factors associated with fire spreading such as humidity, temperature. It is powered by a Machine Learning model that we construct based on all the past forest fires data in California since 2001. With this platform, firefighting forces can better anticipate the severity of a forest fire and thus better manage available resources to combat fires.

Behind the simple click

1. We first scraped data of California wildfire from different reliable websites including University of California Integrated Pest Management Program and California Department of Forestry and Fire Protection, which are presented in different data formats.

2. We then concatenate all data obtained and created a data set with forest fire incidents in California over the past six years using Python.

3. After wrangling the data, we applied principal component analysis (PCA) and built several models predicting the severity of a forest fire, including random forest, polynomial regression, and neural networks.

4. Using log cosh loss function, the neural network was chosen as it has the best overall performance on the test set. We then built the website referencing to the model to visualize our results.
